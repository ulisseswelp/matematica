Resumo: Noções de Lógica

I. Proposição

Uma proposição é uma sentença declarativa que pode ser classificada, sem ambiguidade, como Verdadeira (V) ou Falsa (F). Não são consideradas proposições frases interrogativas, exclamativas ou sentenças abertas. Exemplo: "A Terra é redonda." (V)

II. Negação

A negação de uma proposição P é uma nova proposição, representada por ~P (lê-se "não P"). Ela inverte o valor de verdade da proposição original. Se P é verdadeira, ~P é falsa, e se P é falsa, ~P é verdadeira. Exemplo: Se P é "O céu é azul.", ~P é "O céu não é azul."

III. Proposição Composta - Conectivos

As proposições compostas são formadas pela união de duas ou mais proposições simples através de conectivos lógicos. Os principais são:

  • Conjunção (e): Símbolo ∧. P ∧ Q é verdadeira somente se P e Q forem ambas verdadeiras.
  • Disjunção (ou): Símbolo ∨. P ∨ Q é falsa somente se P e Q forem ambas falsas.
  • Disjunção Exclusiva (ou... ou): Símbolo ⊻. P ⊻ Q é verdadeira quando P e Q têm valores de verdade diferentes.

IV. Condicionais

As condicionais estabelecem uma relação de "causa e efeito" lógica:

  • Condicional (se... então): Símbolo →. P → Q (lê-se "se P, então Q") é falsa apenas quando P (antecedente) é verdadeira e Q (consequente) é falsa. Em todos os outros casos, é verdadeira.
  • Bicondicional (se e somente se): Símbolo ↔. P ↔ Q (lê-se "P se e somente se Q") é verdadeira se P e Q têm o mesmo valor de verdade.

V. Tautologias

Uma tautologia é uma proposição composta que é sempre verdadeira, independentemente dos valores de verdade de suas proposições simples. A última coluna de sua tabela verdade é composta somente por "V". Exemplo: P ∨ ~P

VI. Proposições Logicamente Falsas

Uma proposição logicamente falsa, ou contradição, é uma proposição composta que é sempre falsa. Sua última coluna na tabela verdade é composta somente por "F". Exemplo: P ∧ ~P

VII. Relação de Implicação

Dizemos que a proposição P implica Q (P ⇒ Q) se a condicional P → Q for uma tautologia. Isso significa que a veracidade de P garante a veracidade de Q. É uma relação de consequência lógica.

VIII. Relação de Equivalência

Duas proposições, P e Q, são logicamente equivalentes (P ⇔ Q) se a bicondicional P ↔ Q for uma tautologia. Isso significa que elas possuem a mesma tabela verdade e, portanto, têm o mesmo significado lógico. Exemplo: P → Q ⇔ ~P ∨ Q

IX. Sentenças Abertas, Quantificadores

Uma sentença aberta é uma expressão que contém variáveis e cujo valor de verdade não pode ser determinado. Exemplo: x é um número par. Para se tornarem proposições, elas precisam ser "fechadas" por quantificadores:

  • Quantificador Universal (∀): Lê-se "para todo". Transforma a sentença em uma proposição que afirma que ela é verdadeira para todos os elementos de um conjunto.
  • Quantificador Existencial (∃): Lê-se "existe". Transforma a sentença em uma proposição que afirma que ela é verdadeira para pelo menos um elemento de um conjunto.
