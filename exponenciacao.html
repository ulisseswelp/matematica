Resumo de Exponenciação

A exponenciação, também conhecida como potenciação, é uma operação matemática usada para representar a multiplicação de um número por ele mesmo várias vezes. Ela é composta por dois elementos principais: a base e o expoente.

A forma geral da exponenciação é an, onde:

  • a é a base (o número a ser multiplicado).
  • n é o expoente (o número de vezes que a base é multiplicada).
Exemplo: 23 = 2 x 2 x 2 = 8

Propriedades Fundamentais

As propriedades da exponenciação são regras que facilitam a simplificação e o cálculo de expressões matemáticas.

1. Multiplicação de potências com a mesma base

Mantém-se a base e somam-se os expoentes.

am x an = am+n
Exemplo: 32 x 34 = 32+4 = 36 = 729

2. Divisão de potências com a mesma base

Mantém-se a base e subtraem-se os expoentes.

am ÷ an = am-n
Exemplo: 57 ÷ 53 = 57-3 = 54 = 625

3. Potência de uma potência

Mantém-se a base e multiplicam-se os expoentes.

(am)n = am × n
Exemplo: (23)2 = 23 × 2 = 26 = 64

4. Expoente zero

Qualquer número (diferente de zero) elevado a zero é igual a 1.

a0 = 1 (onde a ≠ 0)
Exemplo: 100 = 1

5. Expoente negativo

Inverte-se a base e muda-se o sinal do expoente.

a-n = 1/an
Exemplo: 4-2 = 1/42 = 1/16
